# Ectocontrol Navien 🤖

Интеграция Home Assistant для управления котлами **Navien** через адаптер **Ectocontrol**.

---

## Особенности ✨

- **Отопление:** Управление режимом работы (ВКЛ/ВЫКЛ) и целевой температурой теплоносителя.
- **Шаг температуры отопления:** Установка температуры кратна 10. Интеграция автоматически округляет некорректные значения.
- **Нагрев ГВС:** Управление режимом работы (ВКЛ/ВЫКЛ) и целевой температурой горячего водоснабжения (ГВС).
- **Датчики (Sensor):** Отслеживание текущих показателей и состояния котла:
  - Температура теплоносителя
  - Температура ГВС
  - Целевая температура теплоносителя
  - Целевая температура ГВС
  - Код ошибки котла
  - Статус котла (название состояния)

---

## Установка 🛠️

Интеграция настраивается через пользовательский интерфейс Home Assistant.

После установки (через HACS или руками):

1. Нажмите кнопку +, ДОБАВИТЬ ИНТЕГРАЦИЮ.
2. Найдите **Ectocontrol Navien**.
3. Введите учетные данные для входа в ваш аккаунт Ectocontrol:
   -  `Email`
   - `Password`
4. После успешной аутентификации введите `object_id` вашего адаптера Navien.

---

## Настройка сущностей ⚙️

После добавления интеграции будут автоматически созданы следующие сущности:

| Домен         | Название по умолчанию      | Описание                                      |
|---------------|----------------------------|-----------------------------------------------|
| `climate`     | Отопление                  | Управление отоплением (ВКЛ/ВЫКЛ, целевая температура) |
| `water_heater`| Горячая вода (ГВС)         | Управление целевой температурой ГВС           |
| `sensor`      | Температура теплоносителя  | Текущая температура в системе отопления       |
| `sensor`      | Температура ГВС            | Текущая температура горячей воды               |
| `sensor`      | Целевая температура теплоносителя | Заданная температура отопления           |
| `sensor`      | Целевая температура ГВС    | Заданная температура ГВС                        |
| `sensor`      | Ошибка котла               | Код ошибки (при наличии)                        |
| `sensor`      | Статус котла               | Текущее состояние котла                         |

### Важные замечания

- **Шаг температуры для отопления:** Целевая температура отопления должна быть кратна 10. Интеграция округляет некорректные значения.
- **Температура теплоносителя при отключенном отоплении:** Если целевая температура из API Ectocontrol равна значению-заглушке **3276.7**, значение температуры будет `None (Неизвестно)`.

---

## Технические детали 💻

- **Платформы:** `climate`, `sensor`, `water_heater`
- **Интервал обновления:** данные обновляются каждые **30 секунд** (`UPDATE_INTERVAL`)
- **Устройство:** все сущности сгруппированы под одним логическим устройством Home Assistant с:
  - **Производитель:** `Ectocontrol` (`DEVICE_INFO_MANUFACTURER`)
  - **Модель:** `Navien Adapter` (`DEVICE_INFO_MODEL`)
